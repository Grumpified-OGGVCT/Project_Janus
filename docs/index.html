⚖ Project Janus

A sovereign, closed-loop archival system — crawl, clone, archive and query the web with a fully local AI brain. No external APIs. No censorship.

Exposes search_archives and view_thread_history. Always opens the vault in read-only mode. Local embeddings via all-MiniLM-L6-v2.

Per-thread crawl with Wayback CDX temporal snapshots. SHA-256 content hashing for deduplication and integrity.

Mirrors an entire domain as navigable .md files. Internal links rewritten to relative paths. Generates a _index.md navigation tree.

Features

  • 🌐 Full-site Markdown clone — every page saved as a linked .md file, navigable in GitHub / Obsidian / VS Code
  • ⏳ Temporal versioning — live, wayback_oldest, and wayback_recent captures side-by-side
  • 🔒 Immutable vault — SHA-256 content hashes; MCP server is permanently read-only
  • 🧲 Semantic search — sentence-transformers + ChromaDB, entirely local
  • 🤖 100% local AI — no OpenAI / Anthropic; all inference via Ollama
  • 🔗 Native tool-calling — Mistral Large 3's function-calling drives the MCP tool loop
  • ✅ CI-tested — unit tests run on every push via GitHub Actions
  • 📡 Org-secret demo — weekly auto-demo via OLLAMA_HOST org secret
  • 🧠 Inner Thoughts — collapsible reasoning trace shows model chain-of-thought and tool decisions
  • 🔄 Self-updating — auto-checks for latest version against main branch via GitHub API

    Quick Start

    1 — Install

    git clone https://github.com/Grumpified-OGGVCT/Project_Janus.git
cd Project_Janus
pip install -r requirements.txt

    2 — Configure targets in main.py

    # Per-thread harvest (adds posts to vault.db):
TARGET_URLS = ["https://your-site.net/threads/example.123/"]
# Full-site Markdown clone (mirrors entire domain):
TARGET_SITE = "https://your-site.net/"
CLONE_MAX_PAGES = 1000

    3 — Run

    # Harvest → clone → launch AI agent:
python main.py
# Agent only (no harvest):
python run_agent.py

    4 — Site clone output layout

    data/site_mirror/your-site.net/
  _index.md               ← navigation tree (every page linked)
  index.md                ← home page
  threads/
    some-thread.md
    another-thread.md
  forums/
    general.md

    5 — Enable this demo page (once)

    GitHub Repo → Settings → Pages
